# 📌 Calculator using Spring Boot

A simple **REST API Calculator** built using **Spring Boot** that performs basic arithmetic operations such as:

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- ✖ Square
- Root

This project demonstrates how to build a **backend REST API using Spring Boot**.

---

# 🚀 Features

✔ Perform basic arithmetic operations  
✔ Simple REST API architecture  
✔ Lightweight Spring Boot application  
✔ Easy to test using browser or Postman

# ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/satwik12dev/Calculator-using-Spring-Boot.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Calculator-using-Spring-Boot
```

### 3️⃣ Run the Application

```bash
mvn spring-boot:run
```

The server will start at:

```
http://localhost:8080
```

# 🌐 API Endpoints

| Operation | Endpoint | Example |
|-----------|----------|--------|
| Addition | `/api/add` | `/api/add?a=10&b=5` |
| Subtraction | `/api/sub` | `/api/sub?a=10&b=5` |
| Multiplication | `/api/mul` | `/api/mul?a=10&b=5` |
| Division | `/api/div` | `/api/div?a=10&b=5` |
| Modulus | `/api/mod` | `/api/mod?a=10&b=5`|
